Rainydays

An outdoor jacket shop designed to make choosing the right rain gear feel simple and confident.

Team: Individual project

Course:JavaScript 1

Focus: Navigation flow, responsive design, interaction patterns

Tools:Figma, HTML, CSS & JavaScript

Link:View in Figma

Project overview

Rainydays is an online shop concept selling durable rain jackets for outdoor adventures. The project began with designing a clickable prototype in Figma, and continued with bringing the interface to life through a fully responsive, multi-page website.

The goal was to translate visual design into clean, semantic HTML and CSS, ensuring consistent layout, clear navigation and accessible interaction across desktop and mobile.

The brief

The prototype was based on a predefined site architecture including a homepage, product listing, product detail page, checkout flow, about and contact pages. The target audience were outdoor-oriented adults aged 30–50.

Visual system & UI design

The visual system was built to create a calm and trustworthy shopping experience. Natural color tones and atmospheric imagery reflect the outdoor brand identity, while clean typography and generous spacing support comfortable reading across pages. A small set of consistent UI components, including buttons, product cards and navigation elements, ensures a cohesive interface throughout the site.

Design decisions for usability

Key interface decisions were guided by making product browsing feel effortless. Large product imagery supports quick visual comparison, while clear typography and spacing make prices, names and call-to-action buttons easy to scan.

Navigation was kept simple and predictable, reducing cognitive load and helping users understand where they are in the site at all times. Primary actions are consistently placed and visually distinct, supporting confident movement toward checkout.

Key screens

The prototype includes a homepage, product list, product detail page and checkout flow. Navigation and interaction patterns were designed to feel predictable and friction-free.

Implementation

The final prototype was developed into a responsive multi-page website using semantic HTML and reusable CSS. Layouts were built with Flexbox and Grid, ensuring accessibility, clean structure and consistent interaction across devices.

To support orientation and reduce navigation friction, a back option was introduced on product listing and product detail pages. The active page is also visually highlighted in the main menu, reinforcing users’ sense of location within the site.
