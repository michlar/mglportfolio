Oslo Science Museum

A modern museum website designed to spark curiosity and engage a younger audience without pandering.

Role: UI Designer

Project type: Individual project

Focus: Responsive layout, accessibility, content hierarchy

Tools: Figma, HTML & CSS

Project overview

Oslo Science Museum is a multi-page website concept created to promote exhibitions and events, provide essential visitor information, and encourage people to visit the museum. The interface was designed to appeal to a younger audience through bold layout choices, strong imagery and dynamic content presentation.

The brief

The task was to build a modern, accessible and responsive website using provided image and text content. The design should appeal to youngsters without feeling childish, and communicate energy and curiosity through layout, typography and imagery.

  • Home page with key information and impactful imagery
  • Exhibitions page with upcoming events and available exhibitions
  • Contribution page (how to contribute)
  • Contact page with hours, address and form
  • Document page (privacy policy + terms)

Structure and layout exploration

I focused on building a clear information hierarchy across pages, making it easy for visitors to find what’s happening now, how to plan a visit, and where to go next. Layout decisions were guided by scanability, predictable navigation and responsive behavior.

Visual system & UI design

The visual system uses bold typography and clear spacing to create a modern editorial feel. Strong imagery adds energy and curiosity, while consistent components keep the interface calm and readable. The style guide helped maintain consistency across pages and screen sizes.

Design decisions for usability & accessibility

The site was designed to support quick scanning and clear orientation. Headings, spacing and contrast help users understand content structure at a glance, while consistent navigation supports predictable movement across pages. Responsive layouts ensure content stays readable without horizontal scrolling.

  • Clear heading hierarchy and readable line length
  • Consistent navigation patterns across pages
  • Accessible form structure and focus states
  • Responsive layouts with Grid/Flexbox and no horizontal scroll

Key screens

The final solution includes a homepage, exhibitions overview, contribution page, contact page and document pages. Each page was designed to feel consistent and easy to navigate, while still highlighting the museum’s energy.

Implementation

The high-fidelity designs were implemented as a responsive multi-page website using semantic HTML and reusable CSS. The project was version-controlled in GitHub and deployed for preview, with progress tracked through a Kanban board.

What I learned

  • Designing modern layouts that stay readable and accessible
  • Creating consistent UI across multiple pages with a style guide
  • Balancing energy and clarity through hierarchy and imagery
